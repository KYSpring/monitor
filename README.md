## 简介  
一个新手入门级别的java web应用  
非常简陋，至于有多简陋，如果你不小心看了，还请不要骂我  
当初实习时在参与一个Java web的项目时写的先导项目，即练手项目  
当时负责前端，所以后端就用java最省事的方式写的……  

##框架  
前端没有用框架，采取最简单的Jquery+ajax方式来进行交互  
UI库使用的是Bootstrap  
后端服务器端使用Java开发  

##地图
